Introduction
===========

This product applies E-Democracy's 2014 style to HTML-formatted email messages
by extendeding the email CSS provided by `gs.content.email.css`_. It is based 
on the style implemented by `edem.skin`_, which is based on designs by
`Tim Leisio`_.
